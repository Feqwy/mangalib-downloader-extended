# mangaLib manga downloader (python) 

Построен на парсинге ответов на запросы к API mangalib.me, собирает полученные изображения в cbz архив (Comic Book Zip Archive), поддерживает асинхронное параллельное скачивание, скачивание диапазона чаптеров, в теории может скачивать 18+ мангу, дополнительно собирает информацию о чаптере внутри cbz архива, при ошибках пытается повторно сфетчить данные с API
